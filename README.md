Week 1:

-Creates, edits, and deletes skills and projects

Setup

This app was written in ruby '2.2.0' and rails '4.2.3'

> bundle install
Create database

> rake db:create
> rake db:migrate
> rake db:test:prepare
Start App:

> rails server
Navigate in your browser to localhost:3000
